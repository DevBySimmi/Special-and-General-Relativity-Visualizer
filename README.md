# 🌌 Special-and-General-Relativity-Visualizer
An interactive Python-based visualization toolkit for Special and General Relativity featuring Lorentz transformations, spacetime diagrams, time dilation, gravitational redshift, orbital precession, and light deflection visualizations.

----base

## 🚀 Features

### Special Relativity
- Lorentz Factor Visualization
- Time Dilation
- Length Contraction
- Spacetime Diagrams
- Energy-Momentum Relation
- Relativistic Kinetic Energy

### General Relativity
- Gravitational Time Dilation
- Gravitational Redshift
- Orbital Precession
- Light Deflection Near Massive Objects

### Visual Features
- Dark Cosmic Theme
- Scientific Plotting
- Multiple Physics Visualizations
- Educational Graphs

## 📊 Physics Concepts Covered

### Lorentz Factor

γ = 1 / √(1 − v²/c²)

### Time Dilation

t = γt₀

### Length Contraction

L = L₀/γ

### Energy-Momentum Relation

E² = (pc)² + (mc²)²

### Gravitational Time Dilation

t₀ = t √(1 − Rs/r)
